# Tienda - Web profesional estática

Este proyecto es una **landing page profesional** para una tienda, pensada como base para personalizar con tu marca, productos reales y textos definitivos.

## Estructura

- `index.html` – Página principal con secciones de inicio, servicios, productos, testimonios y contacto.
- `styles.css` – Estilos modernos y responsive con tipografía *Inter*.
- `script.js` – JavaScript ligero para el menú móvil y el año dinámico del footer.

## Cómo usarla

1. Abre el archivo `index.html` en tu navegador (doble clic o botón derecho → "Abrir con").
2. Edita los textos de la página para adaptarlos a tu negocio.
3. Sustituye los datos de contacto (correo, teléfono, etc.) por los reales de tu tienda.
4. Si quieres desplegarla, puedes subir estos tres archivos a un hosting estático (por ejemplo, Netlify, Vercel o el hosting de tu preferencia).

No necesita dependencias ni instalación de paquetes: es una web estática lista para usarse.
